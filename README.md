========================
Prom chat app
========================


Install project:

* `git clone https://github.com/Eugene-Dobrorodnov/promchat.git`
* `make virtualenv`
* `pip install -r requirements.txt`
* `python create_db.py`
* `python manage.py run.py`

* `Demo: [a https://promchat.herokuapp.com](https://promchat.herokuapp.com/)`
